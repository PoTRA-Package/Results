PoTRA TCGA Pan-Cancer Results: Protein Coding mRNA Mediated Dysregulated Pathways in Cancer

The TCGA pan-cancer analysis utilized the PoTRA algorithm to detect dysregulated pathways from 17 primary sites. Only 7 of 17 primary sites are included in the table below and these are breast, colorectal, kidney, liver, lung, prostate and thyroid. These primary sites were chosen because they had a minimum of 50 samples in normal and case. For more information about the PoTRA algorithm please see the publication. This work was supported by a Seven Bridges Cancer Genomics Cloud Collaborative Project funded by NCI. Please visit Bioconductor for the latest version of PoTRA .
